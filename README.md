# Music

A web-based, network music player, built on Node.js.

## Raspberry Pi Install Script

Download the repo and:

```
sudo ./setup.sh
```

then navigate to `<pi_ip_address>:3000` in a browser.

## Install Development

Download the repo and:

```
npm install
```

### Run

```
npm start
```

and navigate to `<host_ip_address>:3000` in a browser.
